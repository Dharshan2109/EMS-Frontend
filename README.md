# Enterprise Employee Management System (EMS)

## Technology Stack

### Frontend
- React JS
- Axios
- React Router
- Bootstrap
- ChartJS

### Backend
- Java 21
- Spring Boot
- Spring Security
- JWT
- Hibernate
- MySQL

### Features
- JWT Authentication
- Employee CRUD
- Department CRUD
- Search
- Filter
- Pagination
- Sorting
- Analytics Dashboard
- Employee Profile
- Export CSV
- Export PDF
- Swagger
- Docker

```bash
mvn spring-boot:run
